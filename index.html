Настройка

Для настройки сетки необходимо задате некоторве исходные данные в файле самой сетки. После чего она автоматически скомилируется и ее можно использовать как обычный css в проекте.

Для компиляции необходим npm и установленный gulp. Команда для компиляции после выполнения npm i и насторойки сетки gulp.

Структура сетки

Для построения сетки необходимо использовать вложенность данного вида

<div class='flex-row'>
<div class='col-6'>Text or another content</div>
<div class='col-6'>Text or another content</div>
</div>

Типы сеток

flex-row-stick

Имеет отступы для каждой вложенной ячейки

col-1 col-2 col-1 col-3 col-1 col-1 col-1 col-2 col-2

flex-row-stick

Не имеет отступы для первой и последней ячейки. Создает эффект 'прилиания к краям'. Для получения данного эффекта используется увелицение размера обертки и отридцательный margin-left.

col-2 col-3 col-1 col-3 col-3

Выравнивание и позиционированние блоков.

Можно выравнивать блоки внутри Flex-row, с помощь доплонительного класс для контейнера с классои flex-row

Также есть возможность выравнивания по вертикали с помощью добавления align-top/align-center/align-bottom/align-stretch || justify-start/justify-center/justify-end классу flex-row

justify-start

col-3 col-4

justify-center Внимание! выравнивает по центру не совсем по сетке.

col-3 col-4

justify-end

col-3 col-4

Смещение элементов классом offset- (задается блоку col-)

col-1 col-1 col-2 col-3 col-2 col-3

offset-left-4 при нормалоном потоке

offset-left-2 col-2 offset-left-2 col-2

offset-right-4 при орентации контента justify-end

offset-right-2 col-1 offset-right-5 col-3

Особенности

Если размер ячек превышает размер максимальный размер сетки используйте доп. класс wrap (для класса flex-row) который будет осуществлять перенос на новую строку не вмещающиеся блоки.

col-12 col-6

Если необходимо что бы не "влазящий" контент при переносе занимал всю ширину, использовать класс box

col-11 col-1 class Box

Для автоматическо (равномерного) ропределения длинны элемента можно использовать класс .box
